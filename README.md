# 排班助手 (Scheduling Assistant)

## 项目简介
排班助手是一款专为需要倒班工作的人员设计的Android应用。它提供了直观的排班管理、闹钟提醒和数据统计功能，帮助用户更好地管理自己的工作时间。

## 核心功能
- 班次管理：创建、编辑和删除班次
- 闹钟提醒：为不同班次设置闹钟
- 数据统计：查看排班统计和分析
- 个性化设置：自定义班次颜色和提醒方式

### 架构说明
1. **表现层 (UI Layer)**
   - 负责用户界面的展示和交互
   - 采用单Activity多Fragment的结构
   - 使用ViewBinding进行视图绑定
   - 实现Material Design 3设计规范

2. **视图模型层 (ViewModel Layer)**
   - 管理UI相关的数据
   - 处理业务逻辑
   - 通过LiveData实现数据观察
   - 确保配置更改时的数据持久性

3. **数据仓库层 (Repository Layer)**
   - 统一管理数据访问
   - 协调不同数据源
   - 实现数据缓存策略
   - 提供单一数据访问接口

4. **数据源层 (Data Source Layer)**
   - Room数据库：存储班次和闹钟数据
   - SharedPreferences：存储应用配置
   - 文件系统：存储日志和导出数据
   - 远程API：预留云同步接口

### 数据流向
1. UI事件 -> ViewModel
2. ViewModel -> Repository
3. Repository <-> DataSource
4. DataSource -> Repository -> ViewModel -> UI

## 快速开始

### 环境要求
- Android Studio Hedgehog | 2023.1.1 或更高版本
- JDK 17 或更高版本
- Android SDK Platform 34
- Android Build Tools v34.0.0

### 安装步骤
1. 克隆项目到本地
```bash
git clone https://github.com/yourusername/SchedulingAssistant.git
```

2. 使用Android Studio打开项目

3. 同步Gradle依赖
```bash
./gradlew build
```

4. 运行项目
- 选择目标设备（真机或模拟器）
- 点击运行按钮或使用快捷键`Shift + F10`
